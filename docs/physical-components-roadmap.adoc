= Physical Components: Pre-Civilisation Pipeline
:toc: left

Rough design sketch. Not a formal component spec.

NOTE: Urd simulates an arbitrary space body, not Earth. No component assumes Earth-specific conditions. Parameters drive all physical outcomes; Earth is one possible configuration, not the reference.

---

== Generation (one-shot, ordered)

Components run once in sequence. Each bakes frozen state consumed by later stages.

1. **Stellar** — star properties: luminosity, type. Drives all thermal baselines downstream.
2. **Planetary** — advances the planet's celestial position and rotation over time; exposes the orbital and rotational state consumed by all irradiance-dependent components. Rotation direction (prograde/retrograde) is an explicit parameter; it affects Coriolis direction and ocean gyre orientation downstream.
3. **Tectonics (baseline)** — plate seeds, initial crust type (mafic/felsic), fault map, volcanism map.
4. **Elevation** — crustal thickness + noise → absolute elevation field.
5. **Thermal baseline** — per-cell irradiance from stellar output × planetary position × latitude × axial tilt. Static map; no atmospheric redistribution yet.
6. **Hydrology** — total water volume (parameter) distributed across the elevation field → ocean basin assignment, land/ocean mask.

---

== Spin-Up (iterated until stable)

Components run under their natural <<engine-cadence,cadence>> constraints until outputs converge. Cadence differences naturally stagger interdependent components — a slower component will consume the latest state produced by a faster one on its next tick, without requiring explicit coordination. Each component also operates at a defined <<topology-resolution,resolution>> range; coarser resolutions for macro processes, finer for local detail.

NOTE: Tectonics passes survive only if they produce different results from a one-shot baseline; otherwise dropped.

Ordered slowest to fastest <<engine-cadence,cadence>>:

1. **Tectonics (passes)** — geological cadence. Stress accumulation, orogeny, elevation refinement.
2. **Glaciation** _(tentative)_ — orbital/axial cadence. Ice sheet advance and retreat; major driver of erosion and sea level change. Relationship to runtime climate/weather components not yet resolved.
3. **Erosion** — century cadence. Water and ice reshape elevation; feeds back into river network on its next tick.
4. **Biome** — decade cadence. Temperature + moisture + soil → baseline vegetation and soil quality map.
5. **Ocean circulation** — multi-rotational-period cadence. Thermal gradient → surface current equilibrium. Produces warm/cold coastal masks.
6. **River network** — rotational-period cadence. Precipitation → drainage DAG, lake formation, permanent river paths.
7. **Atmospheric circulation** — sub-rotational-period cadence. Coriolis + orography → wind field, rain shadow map, moisture distribution. Owns global atmospheric gas composition (CO2, sulfur particulates); updated each rotational period.

Spin-up ends when all reach a stable fixed point. Runtime inherits that state.

---

== Runtime (continuous)

**Per tick:**

- **Planetary** — advances celestial position and rotation angle. Primary clock for all irradiance-dependent components.
- **Atmosphere** — redistributes heat and moisture; maintains global gas composition pool (CO2, sulfur). Consumes stellar state + planetary state + surface albedo. Produces precipitation field.
- **Hydrology** — propagates precipitation through the river DAG. Produces river flux, lake levels.

**Forcing-triggered only:**

- **Tectonics** — elevation adjustments, volcanic events.
- **River network** — DAG recomputation, triggered by elevation change or drainage threshold crossing.
- **Biome** — rebalances vegetation, triggered by sustained atmospheric or hydrology shift.
- **Isostatic rebound** — elevation adjustment triggered by glacial load change.

---

== Civilisation Handoff

Per-cell state consumed at runtime start:

- Elevation, river flux, soil quality, temperature, precipitation, biome class.

Forcings civilisation may receive:

- Volcanic forcing, flood forcing, drought forcing, glacial forcing.
