= Urd.Engine

The loom — simulation framework.
Defines the contracts and infrastructure that all domain components depend on.
Contains no simulation logic.

== Shafts

|===
|Directory |Purpose

|`Autodoc/` |Attributes that annotate components for documentation generation
|`Clock/` |Simulation clock and tick events
|`Component/` |Base component contract and tick context
|`Data/` |Parameter validation and normalization
|`Inspection/` |Runtime inspection tools — clock compression ratio, scheduler dispatch counters, message bus publish counts
|`Messaging/` |Message bus contract
|`Profiling/` |Per-component tick duration profiling with a rolling window
|`Scheduling/` |Job scheduler
|`Topology/` |Grid abstraction — `IGrid`, `Resolution`, `CellId`
|===
